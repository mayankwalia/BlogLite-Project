BlogLite is a free, online, easy to use blog-sharing application and social network platform.

BlogLite allows users to edit and upload blogs and attach images through a web app. Users can add a caption to each of their blogs. Each post by a user appears on their followers' BlogLite feeds and can be viewed by the public when the blog is not archived. Users also have the option of making their blogs private so that only they can view their blogs and no one else.Users can upload their profile pic and can also add their bio.

As with other social networking platforms, BlogLite users can like, comment on others blogs. Users can connect with each other using the inbuilt keyword based search functionality. They can also download view statistics based on their blogs engagement data along with some nice looking visuals and export it in form of a csv file. The app also provides api support.
